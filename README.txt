Глобальный проект группы студентов GeekBrains курса Университет 2035 

1. Делаем форк (fork) этого проекта.
2. Делаем git clone для вашей версии этого репозитория.
3. Создаете ветку с предлагаемыми изменениями (Например Никнейм или ФамилияИО).
4. Производите все изменения только в этой ветке. Все commit только на АНГЛИЙСКОМ языке!
5. Отправляете эти изменения в свой аккаунт на GitHub (push).
6. В окне на GitHub появляется возможность отправить pull request.
7. Герт А.В. принял участие в глобальном проекте группы студентов GeekBrains курса Университет 2035)
